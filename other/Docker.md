Docker - менеджер контейнеров (как порт). Внутри контейнеров лежат все зависимости, работают на Linux. 
В Docker файле указываем как компилировать наше приложение

Docker compose - собирает кластер, упрощает локальную разработку. Тестирует гипотезы.

Docker образ 

Docker позволяет облегчить сборку приложения, используя образ Linux (Без использования терминала).

Большинство разработчиков собирают приложения через Docker. Сборка через Docker не требует доступ в интернет (так как требует все необходимые библиотеки на системе), и происходит за 10 минут. 
На сервере ничего кроме Linux и Docker.
