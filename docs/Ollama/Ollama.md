Ollama - это простейший способ настроить и запустить большие языковые модели такие как Gemma 3, DeepSeek-R1, Qwen3 и т.д.

# Описание с [Википедии](obsidian://open?vault=Project01&file=resources%2F%D0%98%D1%81%D1%82%D0%BE%D1%87%D0%BD%D0%B8%D0%BA%D0%B8)
Ollama — это программная платформа с открытым исходным кодом для выполнения больших языковых моделей (LLM) локально или в облаке. Она предлагает интерфейс командной строки (CLI) для загрузки, установки, использования и удаления моделей. Предоставляя единый интерфейс для поддерживаемых моделей, она позволяет пользователям использовать модели, с деталями которых они не знакомы. Ollama распространяется под свободной лицензией MIT.
В основе Ollama лежит локальный веб-сервер, предоставляющий унифицированный для всех моделей JSON REST API для управления и взаимодействия с ними. Программа Ollama запускает и управляет этим сервером, а также использует REST API для предоставления интерфейса командной строки для управления моделями и взаимодействия с ними3.
Непосредственно для выполнения моделей Ollama использует движок llama.cpp.
Ollama не имеет собственного GUI, предоставляя пользователю интерфейс командной строки. При необходимости, можно использовать интерфейс сторонних производителей, например Open WebUI.
